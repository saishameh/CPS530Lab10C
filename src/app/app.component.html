Development Process

Installation

To get started with Angular, I installed the Angular CLI globally using the following command:

npm install -g @angular/cli

During installation, I encountered an issue with an outdated version of Node.js. To resolve this, I upgraded Node.js to a version compatible with Angular CLI.

Development

I created this Angular project by running:

ng new my-web-page

After creating the project, I updated the following files:

  • app.component.html: Added the structure and content for this page.
  • styles.css: Customized the styles to enhance the page's appearance.

Challenges and Solutions

Challenge: The Node.js version installed on my system was outdated and incompatible with the Angular CLI, causing issues during the setup and project creation.
Solution: To resolve this, I upgraded Node.js to the latest stable version. This ensured compatibility with the Angular CLI and allowed the installation to proceed smoothly.

Challenge: Implementing form validation was more challenging than anticipated, especially with Angular's reactive forms, as I was not fully familiar with its syntax and structure.
Solution: I researched and watched frew tutorials and was able to apply correct validations and display helpful error messages in real-time.
